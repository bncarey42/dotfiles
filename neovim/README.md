## Installation

To install this config, clone this repo locally and create a symlink to wherever you cloned it
to from `~/.config/nvim`. Alternatively you can clone it directly to `~/.config/nvim` using [sparse git checkout](https://jasonkarns.wordpress.com/2011/11/15/subdirectory-checkouts-with-git-sparse-checkout/)
